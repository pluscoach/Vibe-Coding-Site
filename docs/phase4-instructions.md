## 3. 섹션 3 — 통찰 (#insight)

### 3.1 레이아웃
```
┌──────────────────────────────────────────────────┐
│ (화이트 배경)                                    │
│                                                  │
│   [헤드 — 중앙 정렬, 크게]                       │
│   "대부분의 사업은                               │
│    감으로 시작합니다"                            │
│                                                  │
│   ─────────                                      │
│                                                  │
│   [블록 1 — 메인 본문]                           │
│   배달의민족·메가커피·자본금 카피                 │
│                                                  │
│   ─────────                                      │
│                                                  │
│   [블록 2 — 강조]                                │
│   "AI 구독료만 있어도"                           │
│                                                  │
│   ─────────                                      │
│                                                  │
│   [블록 3 — 클로징]                              │
│   "20% 카피 + 좋은 기획자"                       │
│                                                  │
└──────────────────────────────────────────────────┘
```

### 3.2 헤드 (섹션 최상단)
```
대부분의 사업은
감으로 시작합니다
```
- 폰트: `text-4xl md:text-5xl lg:text-6xl font-bold`
- 색상: `text-slate-900`
- 정렬: `text-center`
- 줄바꿈 위 표기대로 2줄
- 하단 여백: `mb-20`

### 3.3 블록 1 — 메인 본문
```
멋있어 보이는 것을 따라합니다.
그러다 유행에 뒤쳐져 사라집니다.

배달의민족 다음엔 요기요, 쿠팡이츠.
메가커피 다음엔 컴포즈, 빽다방.
같은 구조의 플랫폼이 계속 나옵니다.

이게 가능한 이유는 — 자본금입니다.
```
- 폰트: `text-xl md:text-2xl text-slate-700`
- 줄 간격: `leading-relaxed`
- 정렬: `text-center max-w-3xl mx-auto`
- 단락 사이: `<p>` 3개로 분리, `space-y-6`

### 3.4 블록 사이 구분선
각 블록 사이에 미니멀 구분선:
- `<div class="w-12 h-px bg-slate-300 my-16 mx-auto"></div>`

### 3.5 블록 2 — 강조
```
그런데 디지털 세상에서는
AI 구독료만 있어도
같은 일을 할 수 있습니다.
```
- 폰트: `text-2xl md:text-3xl font-semibold`
- 색상: `text-slate-900`
- 정렬: `text-center max-w-3xl mx-auto`
- "AI 구독료" 부분은 `<span class="text-emerald-600">` 인라인 강조

### 3.6 블록 3 — 클로징
```
새로 창작할 필요 없습니다.
잘 만들어진 상품에
당신의 아이디어 20%를 섞으세요.

AI 시대에 살아남는 사람은
좋은 기술자가 아닌
좋은 기획자입니다.
```
- 폰트: `text-xl md:text-2xl text-slate-700`
- 줄 간격: `leading-relaxed`
- 정렬: `text-center max-w-3xl mx-auto`
- 두 단락으로 분리, `space-y-8`
- "20%"와 "좋은 기획자"는 `<span class="font-bold text-slate-900">` 강조

### 3.7 섹션 패딩
- 섹션: `py-32 md:py-40` (다른 섹션보다 살짝 더 — 카피의 호흡)
- 컨테이너: `max-w-4xl mx-auto px-6 md:px-8`

## 4. 섹션 4 — 비교 (#comparison)

### 4.1 레이아웃
```
┌──────────────────────────────────────────────────┐
│ (슬레이트50 배경)                                │
│                                                  │
│   [섹션 헤드 — 중앙 정렬]                        │
│   "사업은                                        │
│    만드는 것이 아니라                            │
│    팔리는 것을 만드는 것입니다"                  │
│                                                  │
│   [좌우 비교]                                    │
│   ┌────────────────┐  ┌────────────────┐         │
│   │ 기존 방식      │  │ A.MAKERS 방식  │         │
│   │ (붉은톤)       │  │ (민트톤)       │         │
│   │                │  │                │         │
│   │ 유행 발견      │  │ 고객 분석      │         │
│   │   ↓            │  │   ↓            │         │
│   │ 급하게 제작    │  │ 시장 검증      │         │
│   │   ↓            │  │   ↓            │         │
│   │ 고객 없음      │  │ MVP 제작       │         │
│   │   ↓            │  │   ↓            │         │
│   │ 포기           │  │ 출시           │         │
│   │   ↓            │  │   ↓            │         │
│   │ 다시 유행      │  │ 개선           │         │
│   └────────────────┘  └────────────────┘         │
│                                                  │
└──────────────────────────────────────────────────┘
```

### 4.2 섹션 헤드
```
사업은
만드는 것이 아니라
팔리는 것을 만드는 것입니다
```
- 폰트: `text-4xl md:text-5xl font-bold`
- 색상: `text-slate-900`
- 정렬: `text-center`
- 줄바꿈 위 표기대로 3줄
- 하단 여백: `mb-20`
- "팔리는 것" 부분은 `<span class="text-emerald-600">` 인라인 강조

### 4.3 좌우 비교 박스

**그리드:**
- 데스크탑/태블릿: `grid grid-cols-2 gap-6 md:gap-8`
- 모바일: `grid grid-cols-1 gap-4`

**좌측 박스 (기존 방식):**
- 배경: `bg-white`
- 보더: `border border-red-200`
- 상단 강조 보더: `border-t-4 border-red-400` (또는 헤더 부분만 빨강 처리)
- 모서리: `rounded-2xl`
- 패딩: `p-8 md:p-10`

**좌측 박스 헤더:**
```
기존 방식
```
- 폰트: `text-lg font-bold text-red-500`
- 정렬: `text-center`
- 하단 여백: `mb-8`

**좌측 박스 단계 5개 (확정):**
1. 유행 발견
2. 급하게 제작
3. 고객 없음
4. 포기
5. 다시 유행 찾기

**우측 박스 (A.MAKERS 방식):**
- 배경: `bg-white`
- 보더: `border border-emerald-200`
- 상단 강조 보더: `border-t-4 border-emerald-500`
- 모서리: `rounded-2xl`
- 패딩: `p-8 md:p-10`

**우측 박스 헤더:**
```
A.MAKERS 방식
```
- 폰트: `text-lg font-bold text-emerald-600`
- 정렬: `text-center`
- 하단 여백: `mb-8`

**우측 박스 단계 5개 (확정):**
1. 고객 분석
2. 시장 검증
3. MVP 제작
4. 출시
5. 개선

### 4.4 단계 항목 디자인

각 단계 = 텍스트 박스 + 아래 화살표.

**텍스트 박스:**
- 폰트: `text-base md:text-lg font-semibold text-slate-800`
- 패딩: `py-3 px-4`
- 배경: 좌측은 `bg-red-50`, 우측은 `bg-emerald-50`
- 모서리: `rounded-lg`
- 정렬: `text-center`

**화살표 (각 단계 사이):**
- SVG 아래 방향 화살표 또는 `↓` 텍스트
- 색상: 좌측 `text-red-300`, 우측 `text-emerald-300`
- 폰트: `text-2xl`
- 정렬: `text-center`
- 간격: `my-2`

**마지막 단계 뒤에는 화살표 없음.**

화살표 텍스트 사용 시:
```html
<div class="text-center text-2xl text-emerald-300 my-2">↓</div>
```

### 4.5 섹션 패딩
- 섹션: `py-24 md:py-32`
- 컨테이너: `max-w-4xl mx-auto px-6 md:px-8`
