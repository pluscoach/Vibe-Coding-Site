# Phase 3 작업 지시서 — 강사 신뢰 + 페르소나 분석 시스템

## 0. 이번 Phase의 의미

섹션 5(강사 신뢰)와 섹션 6(페르소나 분석 시스템)은 사이트의 차별점 두 축이다.
- 섹션 5: "근데 너 누군데?"에 답하는 신뢰 섹션
- 섹션 6: "다른 강의랑 뭐가 다른데?"에 답하는 차별점 섹션

자산(실제 스크린샷·사진)은 아직 없다. 이번 Phase에서는 **가상 자산으로 형식만 채운다.** 실제 이미지는 Phase 6 폴리싱 단계 또는 별도 자산 준비 후 교체.

## 4. anchor 스크롤 가림 대응

sticky 헤더(64px)로 인해 anchor 클릭 시 섹션 상단이 가려진다. 다음 중 하나 적용:

**옵션 A (권장):** `<html>` 또는 `<body>`에 `scroll-padding-top: 4rem` 한 줄 추가.

**옵션 B:** 각 `<section>`에 `scroll-margin-top-16` Tailwind 클래스 적용.

옵션 A가 코드 깔끔. 옵션 A로 가되, 모바일 헤더는 56px이라 `md:scroll-pt-16 scroll-pt-14` 또는 `scroll-pt-16`로 일괄 처리해도 무방 (8px 차이는 시각적 무해).
